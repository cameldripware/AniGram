# Katkı Rehberi

AniGram projesine katkıda bulunduğunuz için teşekkür ederiz!

## Nasıl Katkıda Bulunabilirim?

### 1. Depoyu Fork Edin

```bash
git clone https://github.com/your-username/AniGram.git
```

### 2. Feature Branch Oluşturun

```bash
git checkout -b feature/amazing-feature
```

### 3. Değişiklikleri Commit Edin

```bash
git commit -m 'feat: Add some amazing feature'
```

### 4. Branch'e Push Edin

```bash
git push origin feature/amazing-feature
```

### 5. Pull Request Açın

Depomuza giderek Pull Request açın ve değişikliklerinizi açıklayın.

## Kodlama Standartları

- **ESLint** ve **Prettier** kullanın
- Anlamlı commit mesajları yazın
- Kodunuzu test edin
- Yorum yazın ve dokumentasyon sağlayın

## Hata Raporlama

Hata bulduysanız, lütfen Issues sekmesinden yeni bir issue açın ve aşağıdaki bilgileri sağlayın:

- Hatanın açıklaması
- Nasıl tekrarlanacağı
- Beklenen davranış
- Gerçek davranış
- İşletim sistemi ve versiyon

## Sorular?

Sorularınız için [cameldripware](https://github.com/cameldripware) ile iletişime geçin.
