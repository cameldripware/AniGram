# AniGram 🎌

**Anime Sosyal Medya Platformu**

AniGram, anime tutkunları için tasarlanmış sosyal medya platformudur. Kullanıcılar anime hakkında paylaşımlar yapabilir, takipçi kazanabilir ve topluluk oluşturabilir.

## 🚀 Özellikler

- ✅ Kullanıcı Kayıt ve Giriş
- ✅ Anime Paylaşımları
- ✅ Takip Sistemi
- ✅ Beğeni ve Yorum Özellikleri
- ✅ Kullanıcı Profilleri
- ✅ Arama Fonksiyonu

## 📋 Gereksinimler

- Node.js v16 ve üzeri
- npm veya yarn
- MongoDB

## 🔧 Kurulum

### 1. Depoyu Klonlayın

```bash
git clone https://github.com/cameldripware/AniGram.git
cd AniGram
```

### 2. Ortam Değişkenlerini Ayarlayın

```bash
cp .env.example .env
```

`.env` dosyasını düzenleyerek gerekli değişkenleri ayarlayın.

### 3. Bağımlılıkları Yükleyin

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

## ▶️ Çalıştırma

### Backend Sunucusunu Başlatın

```bash
cd backend
npm start
```

Sunucu `http://localhost:5000` adresinde çalışacaktır.

### Frontend Uygulamasını Başlatın

```bash
cd frontend
npm start
```

Uygulama `http://localhost:3000` adresinde açılacaktır.

## 🛠️ Teknoloji Stack

- **Frontend:** React, Redux, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js, MongoDB, JWT
- **Veritabanı:** MongoDB
